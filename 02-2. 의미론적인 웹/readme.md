#의미론적인 웹(Semantic Web)

##XHTML->HTML5

###네비게이션
```	
	<h1>JavaScript</h1>
		<nav>
			<ol>
				<li>JavaScript란?</li>
				<li>변수와 상수</li>
				<li>연산자</li>
			</ol>
		</nav>
```

###컨텐츠
```
		<article>
			<h2>변수와 상수</h2>
			변수란?
			<ul>
				<li>변하는 값</li>
				<li>x=10일때 왼쪽항...</li>
			</ul>
		</article>
```


`<header>`, `<nav>`, `<article>` : 실제 화면에서 변하는 것은 없음
