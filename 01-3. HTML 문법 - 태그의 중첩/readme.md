#HTML 문법 - 태그의 중첩

`<li>` : 목록 작성시 각 항목(list item)은 `<li>`와 `</li>`사이에 넣는다.

`<ol>` : 순서 있는 목록(ordered list)을 표시한다

`<ul>` : 순서 없는 목록(unordered list)을 표시한다.

##예시
###`<ol>`
>  <body>
    <ol>
      <li>html</li>
      <li>css</li>
      <li>javascript</li>
    </ol>

###`<ul>`
>    <ul>
      <li>서울대학교</li>
      <li>고려대학교</li>
      <li>연세대학교</li>
    </ul>
